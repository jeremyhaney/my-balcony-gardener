# My Balcony Gardener

ESP32-based smart irrigation system logging temperature, humidity, and watering events. Live dashboard with real-time controls and plotting.

## Features
- DHT11 sensor logging every 5 seconds
- Soil moisture readings (future support)
- Manual “Water Now” trigger
- Local web server with live dashboard
- Graphing via HTML5 Canvas

## Project Goals
- Reliable data logging
- Automated irrigation
- Easy-to-use interface
- Low power consumption
